# 2023-igem-wiki 项目说明

- ## 1. 项目结构说明

  - assets 用来存放图片等素材
  - components 全局组件
  - pages 就是HomeView中router-view渲染内容
  - router 路由
  - store
  - views 暂定只有HomeView

- ## 2. 开发说明

  - **尽量还是不要再在APP.vue写，因为没有设计图，整体框架可能会有变动**
  - 没有原型图，没有设计图，风格样式可以参照往年wiki

- ## 3. 其他说明

  - ### 静态资源上传问题
